CCDroplet
===============================

version number: 0.0.1
author: Ernest Yeung

Overview
--------

BConvective Burning Droplet model for (rocket) Combustion Chambers.

Installation / Usage
--------------------

To install use pip:

    $ pip install ccdroplet


Or clone the repo:

    $ git clone https://github.com/ernestyalumni/ccdroplet.git
    $ python setup.py install
